# House Price Prediction (Linear Regression)

This project implements a Linear Regression model to predict house prices using:

• Square Footage  
• Number of Bedrooms  
• Number of Bathrooms  

## Dataset:
Kaggle - House Prices Advanced Regression Techniques

## Technologies Used:
• Python  
• Pandas  
• Scikit-learn  
• Matplotlib  

## Model Evaluation:
• Mean Squared Error (MSE)  
• R² Score
